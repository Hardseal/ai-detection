# 3.1.1: Access Control Narrative

Access to CUI-bearing systems at Hardseal is controlled through Okta Single Sign-On with Conditional Access policies. All administrators authenticate using YubiKey FIDO2 hardware tokens; no TOTP fallback is permitted on privileged accounts. The ISSO reviews all access entitlements every 90 days, referencing the access-review log stored in Splunk index `auth_audit`.

Exceptions require written approval from the CISO and expire after 90 days. Terraform manages all Okta group memberships under version control; changes are peer-reviewed and applied via GitHub Actions with signed commits. BitLocker enforces at-rest encryption on all Windows endpoints; FileVault on macOS.

Responsible role: ISSO. Review cadence: quarterly. Evidence: Splunk search `index=auth_audit sourcetype=okta`, exported monthly as CSV to the evidence vault.
