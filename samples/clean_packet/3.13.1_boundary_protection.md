# 3.13.1: Boundary Protection Narrative

The CUI enclave boundary is enforced by a Palo Alto PA-3220 firewall pair configured in HA active/passive. All inbound traffic terminates at a WAF layer before reaching application services; east-west traffic crosses micro-segmentation rules validated against the threat model quarterly.

TLS 1.3 is mandatory for all external connections; TLS 1.2 is permitted only for legacy vendor APIs that cannot be upgraded, tracked as POA&M item PA-013. Egress DNS is forced through Cloudflare Gateway with category-based blocking. CrowdStrike Falcon runs on every endpoint and reports to the Splunk SIEM with SHA-256 event integrity.

Firewall rule changes flow through a Terraform pull-request workflow; no manual console changes are permitted. The sysadmin and ISSO both approve every merge. Review cadence: monthly rule audit, quarterly architectural review.
